函数返回值

	


1)用全局变量得到函数中要返回的值

2)用return value 方式
  $?是shell变量,表示"最后一次执行命令"的退出状态 0为成功,非0为失败.
	在函数中,可以用return设置返回状态,return后跟数值n(0-255),n不设置则默认为0;
			havefun () {
			  echo 'fun!'
			  return 123
			}
			havefun
			RET=$?
			
3)用 echo 法
  在函数的最后,用echo命令把变量输出到stdio中，调用函数时格式：ret(变量名)=`function_name(函数名)`
  注意: function_name 一定要用反引号引起来, 表示命令的执行
  例如：
  		#!/bin/sh
  		getAnswer() {
  			echo 'answer'
  		}
  		
  		ret=`getAnswer`
